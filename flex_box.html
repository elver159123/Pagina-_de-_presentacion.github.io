Flexbox y CSS Grid: Tu Guía Rápida

Entendiendo las herramientas clave para layouts en CSS

Flexbox: Diseño en una Dimensión (Filas o Columnas)

Flexbox es ideal para alinear elementos en una sola dimensión, ya sea en una fila o en una columna. Piensa en él para barras de navegación, componentes de tarjetas o distribución de elementos dentro de un área más pequeña.

Ejemplo 1: Centrar un elemento

Centro

CSS: display: flex; justify-content: center; align-items: center;

Ejemplo 2: Distribución de elementos

Item 1
Item 2
Item 3

CSS: display: flex; justify-content: space-around;

Ejemplo 3: Orden de los elementos

Primero
Segundo
Tercero

CSS: order: [número]; (aplicado a los ítems flex)

---

CSS Grid: Diseño en Dos Dimensiones (Filas y Columnas)

CSS Grid te permite diseñar layouts complejos en dos dimensiones, gestionando tanto filas como columnas a la vez. Es perfecto para la estructura general de tu página, un dashboard o un diseño de galería de imágenes.

Ejemplo 1: Layout básico de 3 columnas

Header
Sidebar
Contenido Principal
Footer

CSS: display: grid; grid-template-columns: 1fr 2fr 1fr; grid-template-rows: auto 1fr auto;

Ejemplo 2: Uso de `grid-template-areas`

Encabezado
Navegación
Contenido
Publicidad
Pie de Página

CSS: grid-template-areas: "header header" "nav main" "nav aside" "footer footer";

Ejemplo 3: Colocar elementos con líneas (line-based placement)

Item A
Item B (2 columnas)
Item C

CSS: grid-column: 1 / 3; (para Item B)
