¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para estructurar páginas web. Utiliza etiquetas para definir elementos como títulos, párrafos, imágenes, enlaces y más.

<!DOCTYPE html>
<html>
  <head>
    <title>Mi primera página</title>
  </head>
  <body>
    <h1>¡Hola Mundo!</h1>
    <p>Este es un párrafo en HTML.</p>
  </body>
</html>

¿Qué es CSS?

CSS (Cascading Style Sheets) se utiliza para aplicar estilos a las páginas web, como colores, tamaños de texto, márgenes, alineaciones y más.

body {
  background-color: #f4f4f4;
  font-family: Arial, sans-serif;
}

h1 {
  color: #333;
}

¿Qué es CSS Grid?

CSS Grid Layout es una técnica avanzada de diseño que permite organizar elementos en filas y columnas de forma sencilla y flexible.

.contenedor {
  display: grid;
  grid-template-columns: 1fr 2fr;
  gap: 20px;
}

Con CSS Grid puedes crear diseños adaptables y organizados para tus páginas web modernas.
